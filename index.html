Bitácora Segunda y Tercera unidad

Semana 1: Inicio de Unidad

Comenzamos la unidad con la introducción a nuevos proyectos en el laboratorio de computación. Organizamos nuestros archivos y establecimos las metas de aprendizaje para este bimestre, enfocándonos en mejorar nuestras habilidades de diseño web.

Semana 2: Proyectos y Laboratorios

Durante esta semana trabajamos en la estructura avanzada de HTML. Fue una jornada intensa de práctica en clase, resolviendo dudas y aplicando estilos CSS para que nuestras páginas del colegio se vean mucho más profesionales y creativas.

Semana 3: Simulación en Tinkercad

En esta sesión construimos un circuito interactivo utilizando una placa Arduino Uno, una placa de pruebas (breadboard), resistencias y luces LED. Añadimos el código de programación para controlar el encendido secuencial de los componentes.

Semana 6: Sensor de Distancia Ultrasónico

En esta práctica realizamos la simulación en Tinkercad de un circuito con Arduino Uno y un sensor ultrasónico de distancia (HC-SR04). Programamos el componente para detectar objetos dentro de un rango determinado y calcular la distancia en tiempo real durante la simulación.
